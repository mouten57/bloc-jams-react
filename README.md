### Description

- Classic STREAM is a music-streaming app built using `React`, `react-router`, and `react-bootstrap`.

## Features

- Styled using react-bootstrap
- App routed using `react-router`

## How to use

1. Clone
2. Open
3. run `npm install`
4. run `npm start`
